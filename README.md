Student Performance Dashboard using Power BI

Project Overview

This project presents an interactive Student Performance Dashboard developed using Microsoft Power BI. The dashboard analyzes student performance based on marks, attendance, subjects, and other relevant factors to support data-driven insights.

Objectives • Identify top-performing students • Analyze subject-wise performance • Evaluate the impact of attendance on marks • Visualize the distribution of student marks • Enable interactive filtering for better analysis

Dataset Information

The dataset consists of 100 student records with the following fields: • StudentName • Class • Subject • Marks • Attendance • Gender • ExamDate • Teacher

Dashboard Features

Key Performance Indicators (KPIs) • Average Marks • Average Attendance (%) • Total Students

Visualizations • Bar Chart: Average Marks by Subject • Bar Chart/Table: Student Ranking • Scatter Chart: Attendance vs Marks • Column Chart/Histogram: Marks Distribution • Pie Chart: Gender Distribution • Line Chart: Marks Over Time

Filters (Slicers) • Class • Subject • Teacher • Gender

Tools and Technologies • Microsoft Power BI • Data Modeling using DAX • Data Visualization Techniques

Key Insights • Higher attendance generally correlates with better academic performance • Certain subjects have lower average scores, indicating higher difficulty • The dashboard helps identify top-performing students efficiently • Marks distribution provides an overview of overall student performance

Design Considerations • Clean and structured layout • Consistent color scheme • Proper alignment and spacing • Interactive and user-friendly interface

How to Use 1. Open the .pbix file in Power BI Desktop 2. Use the slicers to filter data 3. Interact with the visuals to explore insights

Conclusion

This dashboard provides a structured and interactive approach to analyzing student performance. It enables educators and stakeholders to identify trends, evaluate performance, and make informed decisions.
